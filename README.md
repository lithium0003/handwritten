# handwritten
This is source files of iOS app [漢字の書き取り](https://itunes.apple.com/jp/app/id1569208844?mt=8).

## 説明
iPadでApple Pencilを使って、漢字やひらがなの文字の画像を作成し、機械学習のデータセットを作ることができます。
表示される文字と同じ文字を手書きすると、iCloudの文字コード毎のフォルダに保存されていきます。
